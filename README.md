# 🎤 AI Interviewer Bot

An AI-powered Interview Preparation Assistant built using Streamlit.

## 🚀 Features
- Technical & HR interview questions
- Voice input support
- AI-based answer evaluation
- Score & feedback system
- Real-time response generation

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API
- SpeechRecognition
- Pyttsx3

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run Main.py
